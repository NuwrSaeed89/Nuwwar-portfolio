# CV PDF fonts

- **`Cairo-Variable.ttf`** — [Cairo](https://fonts.google.com/specimen/Cairo) from [Google Fonts](https://github.com/google/fonts), licensed under the [SIL Open Font License](https://openfontlicense.org/).

This single variable font is used for Arabic PDFs so **Arabic and Latin** (emails, URLs, company names) render from one file.

English PDFs use the built-in **Helvetica** fonts embedded in PDF (no custom files, maximum compatibility with `@react-pdf`).
