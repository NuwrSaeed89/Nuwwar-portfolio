# Nuwwar Saeed — WordPress Developer Portfolio

A Next.js portfolio site for **Nuwwar Saeed**, WordPress developer.

## Stack

- **Next.js 14** (App Router)
- **TypeScript**
- **Tailwind CSS**

## Run locally

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Build for production

```bash
npm run build
npm start
```

## Customize

- **Contact**: Update the email and LinkedIn URL in `src/components/Contact.tsx`.
- **Projects**: Edit the `projects` array in `src/components/Projects.tsx` to add real work.
- **About / Skills**: Adjust copy and skills in `src/components/About.tsx` and `src/components/Skills.tsx`.
